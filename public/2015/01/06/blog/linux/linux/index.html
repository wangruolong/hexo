Nginx

安装

  1. 需要先安装Nginx的依赖。-y是可选参数，意思是在安装的过程中遇到Is this OK[y/d/N]的时候，自动选择y。这个其实就是和Windows环境下的默认安装是同一个道理。

  2. 下载Nginx的安装包。

  3. 解压安装包。

  4. 运行configure。参数--with-http_ssl_module顾名思义，就是准备等下安装Nginx的时候带上ssl模块。

  5. 编译。
    make

  6. 安装。
    make install

需要特别注意的是alias后的的路径需要以/结尾
